# neo
neo is a fuzzy finder inspired by fnf and supports basic colours.

# Usage
```
cat file | neo
echo "choice1\nchoice2\nchoice3" | neo
```

# Dependencies
None

# Building
You will need to run these with elevated privilages.
```
$ make
# make install
```
